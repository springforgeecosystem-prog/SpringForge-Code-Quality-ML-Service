Option 1: Manual Testing in Postman

Copy any payload from the JSON artifact

POST to http://127.0.0.1:8081/predict-antipattern

Save the Python script as test_antipatterns.py
Make sure your FastAPI server is running

Check test_results.txt for detailed report

venv\Scripts\activate

python test_antipatterns.py
